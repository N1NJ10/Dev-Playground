# TXT Directory

This directory contains supporting text assets, mostly related to secret or credential discovery workflows.

## Files

- `searchstrings.txt`: a categorized keyword list containing common secret, password, token, configuration, and credential-related terms.
- `Secret-Regex`: a collection of regex patterns for identifying common API keys, tokens, private keys, database connection strings, JWTs, and password-like values.
- `Dorks` : various search queries for sensitive data.
