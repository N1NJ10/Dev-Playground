# PowerShell Directory

This directory contains PowerShell tooling scripts.

## Files

- `PSPW/`: an in-memory WMI-based Windows process monitor for local or remote process creation visibility, with polling, extrinsic, and hybrid modes.
- `Invoke-WindowsSearch/`: a Windows Search index querying tool for local or remote keyword hunting, with support for wordlists, filtering, and structured output formats.
