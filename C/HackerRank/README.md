# HackerRank C Exercises

This directory contains small C programs written as solutions to HackerRank problems.

## Files

- `Functions_in_C.c`: practice with function definitions, calls, and basic logic.
- `Hello World.c`: a simple introductory program that prints output.
- `If_Statements.c`: basic conditional branching using `if` and related control flow.
- `Pointers_In_C.c`: pointer-focused exercise, likely covering addressing and dereferencing.
