# C Directory

This folder contains C practice work split into two groups:

- `HackerRank/`: small solutions for beginner-to-intermediate coding exercises.
- `Learning/`: standalone experiments for core C concepts such as pointers, memory, arrays, structs, and file handling.

## Subdirectories

- `HackerRank`: challenge-style programs written as exercise solutions.
- `Learning`: concept-focused examples and a small reusable subproject named `Small-Encoder`.
