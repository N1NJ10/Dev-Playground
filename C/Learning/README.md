# C Learning Notes And Practice

This directory contains standalone C files used to practice specific language features and low-level concepts.

## Files

- `Arrays_Input_Output.c`: reading from and writing to arrays.
- `Conditions.c`: conditionals and comparison-based logic.
- `Datatypes.c`: basic C data types and how they behave.
- `Double-Pointers.c`: an introduction to pointers to pointers.
- `Double-Pointersv2.c`: a second pass on double-pointer usage.
- `Double-Pointersv3.c`: a third variation exploring double pointers in more detail.
- `Even_Number.c`: a small program for checking or handling even numbers.
- `FileDescriptors.c`: practice around file descriptors and lower-level I/O ideas.
- `Functions.c`: basic function declaration and function call examples.
- `FunctionsV2.c`: a follow-up file with additional function practice.
- `Initialization_vs_declaration.c`: distinguishes declaring a variable from initializing it.
- `Loops.c`: loop constructs such as `for`, `while`, or `do-while`.
- `malloc.c`: dynamic memory allocation using `malloc`.
- `malloc_v2.c`: a second dynamic allocation example.
- `Memory_Leaks.c`: explores what causes memory leaks and how to avoid them.
- `MetaData.c`: likely an experiment around descriptive information attached to data or files.
- `Pointers.c`: general pointer practice.
- `Pointers-VeryGoodExample.c`: a more complete pointer example for learning/reference.
- `Pointers_V0.c`: an early/basic pointer exercise.
- `Pointers_V1.c`: a follow-up pointer exercise.
- `Scope.c`: variable scope and lifetime rules.
- `Static_Alloc.c`: static allocation examples.
- `Static_Mem_Alloc.c`: static memory allocation practice.
- `Strings.c`: C string handling and manipulation.
- `Structure.c`: struct definition and field access.
- `Structures_V2.c`: another struct-focused example with expanded usage.
- `arrays.c`: general array practice.

## Subdirectories

- `Small-Encoder`: a more organized mini-project showing XOR string obfuscation plus Base64 encode/decode support.
